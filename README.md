# GlassChat

یک اپ چت محلی (بدون اینترنت) که دو گوشی را از طریق هات‌اسپات/وای‌فای مشترک
به هم وصل می‌کند، با ظاهری شیشه‌ای الهام‌گرفته از iOS 26 (Liquid Glass) و
حس‌وحال آخرین نسخه‌ی تلگرام.

## چطور اجرا کنم؟

1. [Android Studio](https://developer.android.com/studio) رو نصب کن (آخرین نسخه).
2. پوشه‌ی `GlassChat` رو با گزینه‌ی **Open** در Android Studio باز کن.
3. صبر کن Gradle خودش Sync بشه (دفعه‌ی اول wrapper و وابستگی‌ها رو دانلود می‌کنه، به اینترنت نیاز داره).
4. روی هر دو گوشی، یکی هات‌اسپاتشو روشن کنه و گوشی دوم بهش وصل بشه (یا هر دو به یک وای‌فای وصل باشن).
5. اپ رو با دکمه‌ی ▶️ Run روی هر دو گوشی نصب و اجرا کن.
6. توی هر دو اپ، از آیکون شخص‌بعلاوه (بالای صفحه چت) وارد «اتصال به گوشی دیگر» شو؛
   اپ خودکار دستگاه‌های دیگه روی همون شبکه رو پیدا می‌کنه. روی یکی بزن تا چت وصل بشه.

## ساختار پروژه

- `MainActivity.kt` — ناوبری بین صفحه‌ی چت و صفحه‌ی کشف دستگاه
- `ui/screens/ChatScreen.kt` — صفحه‌ی چت با حباب‌های شیشه‌ای
- `ui/screens/DeviceDiscoveryScreen.kt` — اسکن شبکه‌ی محلی و لیست گوشی‌های پیدا‌شده
- `ui/components/GlassComponents.kt` — کامپوننت‌های شیشه‌ای قابل‌استفاده مجدد
- `network/ChatConnection.kt` — سوکت TCP برای ارسال/دریافت پیام
- `network/NetworkScanner.kt` — اسکن رنج آی‌پی شبکه‌ی محلی برای پیدا کردن گوشی‌های دیگر

## نکات فنی

- ارتباط کاملاً محلی و آفلاین است (بدون سرور، بدون اینترنت) — از سوکت TCP خام روی پورت `8988` استفاده می‌شود.
- کشف دستگاه با اسکن موازی رنج `192.168.x.1` تا `192.168.x.254` روی همون ساب‌نت هات‌اسپات/وای‌فای انجام می‌شود.
- طراحی شیشه‌ای بدون وابستگی خارجی و فقط با ترکیب گرادیان + شفافیت + بوردر پیاده شده تا روی همه نسخه‌های اندروید (از API 24 به بالا) یکسان و پایدار نمایش داده شود.

## قابل توسعه

- افزودن آیکون واقعی اپ (فعلاً یک آیکون ساده‌ی وکتور گذاشته شده)
- پشتیبانی از ارسال عکس/فایل
- تاریخچه چت (ذخیره‌سازی محلی با Room)
- افکت بلور واقعی پشت زمینه (RenderEffect) برای API 31+
